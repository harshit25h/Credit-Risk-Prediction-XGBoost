# Credit Risk Prediction using Logistic Regression and XGBoost

## Objective

Predict customer creditworthiness using machine learning techniques and identify the key factors influencing credit risk.

---

## Dataset

German Credit Dataset

- 1000 customer records
- 20 predictor variables
- Binary target:
  - Good Credit
  - Bad Credit

---

## Tools Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Matplotlib

---

## Data Preprocessing

- Data Cleaning
- One-Hot Encoding
- Train-Test Split

---

## Models Implemented

### Logistic Regression

Accuracy: 79%

### XGBoost

Accuracy: 77.5%

### 5-Fold Cross Validation

Average Accuracy: 74.7%

---

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Classification Report
- Cross Validation

---

## Top Predictive Features

1. Credit History
2. Status of Checking Account
3. Foreign Worker Status
4. Loan Purpose
5. Installment Plan Information

---

## Key Findings

- Logistic Regression outperformed untuned XGBoost on this dataset.
- Credit history and account status were the strongest predictors of creditworthiness.
- Cross-validation provided a more reliable estimate of model performance.
